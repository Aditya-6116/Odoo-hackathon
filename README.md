# AssetFlow

## Project Overview

AssetFlow is a React, Vite, Tailwind CSS, and Firebase application scaffold for managing organizational assets, allocation, booking, maintenance, reports, and notifications.

## Folder Structure

```text
src/
  assets/
  components/
  pages/
  routes/
  firebase/
  services/
  hooks/
  contexts/
  utils/
  constants/
  types/
  styles/
```

## Tech Stack

- React with Vite
- React Router
- Tailwind CSS
- Firebase
- JavaScript

## Development Phases

1. Project setup and architecture
2. Authentication and routing
3. Core asset modules
4. Reporting and notifications
5. Testing, polish, and deployment

## Modules

- Authentication
- Dashboard
- Organization
- Assets
- Allocation
- Booking
- Maintenance
- Reports
- Notifications
- Profile

## How to Run

Install dependencies and start the Vite development server when implementation begins.

```bash
npm install
npm run dev
```

## Future Scope

- Firebase authentication
- Firestore data modeling
- Role-based access control
- Asset lifecycle workflows
- Reporting dashboards
- Notification system
